# Proyek Machine Learning: Clustering dan Klasifikasi

Selamat datang di repository proyek Machine Learning ini! Proyek ini mencakup dua tahap utama: **Clustering** untuk mengelompokkan data dan **Klasifikasi** menggunakan hasil clustering sebagai label untuk model klasifikasi. Proyek ini dibangun sesuai dengan kriteria yang telah ditetapkan, dan berhasil memenuhi setiap langkah yang disyaratkan.

## Deskripsi Proyek
Proyek ini bertujuan untuk membangun model machine learning yang memanfaatkan metode **Clustering** untuk pengelompokan data dan **Klasifikasi** untuk memprediksi label berdasarkan hasil clustering. Setiap langkah dalam proyek ini mengikuti template dan instruksi yang telah diberikan, memastikan bahwa setiap bagian dilaksanakan dengan benar.
